## 4. The Value Ladder: Free Visibility -> Paid Control

### **Tier 1: Document Insights (Free Forever)**
*   **"The Mirror"**: Instantly see your document mess.
    *   **Storage Analysis**: Visual heatmaps of where your storage is going.
    *   **Stale File Detection**: Identify files untouched for 3+ years.
    *   **Basic Risk Scan**: See how many files are "Public on the Web" vs "Internal Only".

### **Tier 2: Lifecycle Management (The USP - Paid)**
*   **"The Engine"**: Automated governance for high-stakes industries.
    *   **Automated Retention Policies**: Set "set-and-forget" rules. *Example: "Auto-archive all client contracts 7 years after 'Closed' status."*
    *   **Legal Hold Workflows**: Freeze specific document sets instantly for litigation or audit, preventing any deletion or modification.
    *   **Role-Based Access Enforcement**: Automatically revoke access when an employee leaves or changes departments.
    *   **Lifecycle States**: Define custom document stages (Draft -> Review -> Signed -> Archived) and automate permissions at each stage.
    *   **Compliance Vault**: Immutable logs of every view, share, and edit for HIPAA/GDPR audits.
